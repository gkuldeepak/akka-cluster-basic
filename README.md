# Akka-Cluster-Basic
This template is all about setting and running up your first Akka Clusters using Akka Actors in Scala.

## This template uses - 
- Akka Cluster
- Akka Actor
- Actor Logging

## Getting Started - 
- Use 'sbt clean compile' to compile the project.
- Execute the project using 'sbt run'.

## Pre-Requisites - 
- Java
- Scala
- Akka
